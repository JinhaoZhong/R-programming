# Getting and Cleaning Data Course Project
## Files
- 'README.md': explains the project file structure
- 'CodeBook.md': describes the variables, the data, and the transformations performed
- 'run_analysis.R': merges the training and the test sets, extracts the measurements on the mean and std, creates a tidy data set
- data/*: contains the [data](https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip) for the project
- 'result.txt': contains the tidy data set
